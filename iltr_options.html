Bluesky Inline Translator

Bluesky's default translation feature requires leaving the page to use Google Translate. This extension provides seamless inline translation using Chrome's built-in AI translator.

You can translate both when reading and writing posts. For languages other than English, Spanish, and Japanese, you can use the external Gemini API.

⚠️ Required Setup for Local Translation

  1. Open chrome://flags/#translation-api in a new tab
  2. Find "Experimental translation API"
  3. Change the setting to "Enabled"
  4. Restart Chrome to apply changes
